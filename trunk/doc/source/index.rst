Getting help
============
Having trouble? We'd like to help!

* Try the :doc:`FAQ <faq>` -- it's got answers to some common questions.
* Looking for specific information? Try the :ref:`genindex` or :ref:`modindex`.
* Search for information or ask question in `the PyV8 mailing list`_
* Report bugs with Scrapy in our `ticket tracker`_.

.. _the PyV8 mailing list: https://groups.google.com/group/pyv8/
.. _ticket tracker: http://code.google.com/p/pyv8/issues/list
